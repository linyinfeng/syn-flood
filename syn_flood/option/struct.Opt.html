Fields

debug: bool

Activate debug mode.

Enabling this option will set the log level of the tool to Trace. To set the log level of other modules used by the tool, please use RUST_LOG environment variable.

destination: String

Destination address and port.

Including an address and a port. If a domain is provided, the tool will resolve it.

ipv4: bool

Force using IPv4 address.

This option conflicts with option --ipv6.

ipv6: bool

Force using IPv6 address.

This option conflicts with option --ipv4.

buffer_size: usize

Sets buffer size.

ttl: u8

Sets ttl of IP packets.

window_size: u16

Sets window size of TCP packets.

min_port: u16

Sets minimal port number.

max_port: u16

Sets maximal port number.

source_address: Option<IpAddr>

Force using source address.

source_port: Option<u16>

Force using source port.

output_interval: Duration

Sets output interval(seconds).

number: Option<usize>

Number of packets to be sent.

time: Option<Duration>

Maximal time of sending packets.
